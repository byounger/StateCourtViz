National Center for State Courts Viz-a-thon Visualization Project By Brittney Younger
OVERVIEW

Inspired by the National Center of State Courts' (NCSC) existing court diagrams, I wanted to primarily focus my visualization on the process of searching and learning about a case’s appeal process. The visualization is meant to be exploratory at first and gradually become an explanatory snapshot of a specific appeal process. Each step a user takes gets he or she closer to understanding a state’s court system, what type of cases and what type of courts that are associated with a selected state.

Objective 1: Users can look up states through the search navigation bar at the top of the map, select states based on certain population groups, search for neighboring states and compare.

Objective 2: Users can review and learn about a selected state’s court structure and possible cases.

APPROACH

Based on the timeline of developing the visualization and existing data structure, I decided to build my visualization using Tableau Public. It is a great tool that allowed me to import and sync specific Microsoft Excel tables, add quick detailed tooltips and create unique displays. In addition for the map portion of the visualization, I used Mapbox for the map layout.

For my visualization website layout, I decided to use Bootstrap for my foundation. Bootstrap provides great out-of-the-box solutions and navigation options.

For my website host, I decided to deploy my website using Github, which is another great open source resource for hosting files, data, and code.
